🚀 Karo Pitch

Karo Pitch is a modern, animation-driven startup showcase platform designed to simulate a real investor–startup ecosystem. It focuses on smooth UI, engaging interactions, and a premium user experience.

✨ Features

🎯 Hero Section Animation – Smooth scroll-based animation inspired by modern SaaS websites

📊 Live Counter Stats – Dynamic counters for startups, investors, and funding

🧾 Startup Cards – Interactive and animated startup showcase cards

🎥 Pitch Modal Popup – Click on startups to view pitch previews

💬 Testimonials Slider – Auto-scrolling investor feedback section

🎨 Modern UI/UX – Clean, responsive design with attention to detail

🌙 Dark/Light Mode – Theme switching support

⚡ Smooth Animations – Scroll reveal and micro-interactions

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Styling: Tailwind CSS, Bootstrap

Backend (basic): PHP

Database: MySQL

Other: Shopify (for e-commerce integrations if applicable)
